# X Layer Treasure Hunter - 宝箱稀有度评级指南

本 Skill 使用以下标准对 X Layer Uniswap 流动性池或其他 DeFi 机会进行“宝箱”稀有度评估。评估结合 **Onchain OS Skill**（市场数据、代币发现）和 **Uniswap AI Skill**（路径优化、流动性分析）自动计算。

## 稀有度等级定义

| 等级       | 名称         | 描述                                   | 典型条件（参考值）                          | 颜色/表情建议 |
|------------|--------------|----------------------------------------|---------------------------------------------|---------------|
| N          | 普通宝箱     | 基础机会，收益一般                     | APY < 8% 或流动性较浅                       | ⚪            |
| R          | 稀有宝箱     | 值得关注的中等机会                     | APY 8-15%，中等流动性，滑点可控            | 🟢            |
| SR         | 超级稀有     | 高潜力机会                             | APY 15-30%，良好流动性，低到中滑点         | 🔵            |
| SSR        | 闪耀级       | 极具吸引力的高收益机会                 | APY > 30%，优秀流动性，低滑点，市场热度高  | 🟣            |
| 传说级     | 传说级宝箱   | 罕见的高回报 + 低风险组合              | APY > 50% 或极低风险 + 高流动性 + 额外激励 | 🏆✨           |

## 评估核心指标（优先级从高到低）
1. **真实预期 APY**（考虑 impermanent loss、Gas 费后）
2. **流动性深度**（TVL > $500k 视为良好）
3. **预期滑点**（大额交易滑点 < 0.5% 为佳）
4. **风险评分**（波动率、项目背景、锁仓时间）
5. **市场热度**（Onchain OS token discovery 热度、交易量）

## 使用说明
- Skill 会自动调用 Onchain OS + Uniswap AI 计算以上指标。
- 发现新机会时会明确标注稀有度，例如：“一个闪着金光的 **SSR 闪耀级宝箱** 出现了！”
- 用户可随时询问“这个宝箱是什么等级？”来触发详细评估。

**注意**：所有评级仅供参考，链上收益存在风险，执行交易前必须二次确认。
